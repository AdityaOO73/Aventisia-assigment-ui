# KnowledgeHub UI 🚀

Responsive Knowledge Base UI built with React and Tailwind CSS, featuring reusable components, clean architecture, and modern design.

---

## 📌 Project Overview

This project is a pixel-accurate implementation of a Knowledge Base dashboard based on a Figma design.

It focuses on:
- Clean UI implementation
- Scalable component structure
- Responsive design across all devices
- Maintainable and readable code

---

## 🎯 Features

- Fully responsive (Mobile, Tablet, Desktop)
- Sidebar with adaptive behavior
- Header with search, notifications, and workspace dropdown
- Card-based layout
- "Create New" modal with form validation
- Clean input UI (light grey borders)
- Reusable components
